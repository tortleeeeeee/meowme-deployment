# 🐱 MeowMe - Laravel Blog Application

A simple blog web application built with Laravel where users can register, log in, and share posts publicly.

---

## 🚀 Features

- User registration and authentication (Laravel Breeze)
- Create, edit, and delete blog posts
- Public post viewing
- User-specific post management
- Responsive UI using Bootstrap

---

## 💡 Tech Stack

- **Laravel** – Backend framework
- **Laravel Breeze** – Authentication system
- **MySQL** – Database management
- **Bootstrap** – UI styling

---

## 📌 Purpose

This project was built to practice CRUD operations, authentication, and basic backend development using Laravel.
